# FolodyOS
FolodyOS is a simple, semi-rolling Arch-based Linux distro made by Folody Community (or Folody Team, idk).

![img](https://i.imgur.com/LRxYXoc.png)

# Packages
- DE: customized GNOME
- Browser: Firefox
- Terminal: GNOME Terminal, Alacritty
- File manager: Nautilus
- Video player: mpv
- AUR Helper: yay
